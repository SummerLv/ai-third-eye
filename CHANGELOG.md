# Changelog

All notable changes to this project will be documented in this file.

## [v1.8.4] - 2026-04-30

### Added
- 🔊 新增「开始对话音效」反馈 - 更有仪式感的对话启动体验
  - 使用 Web Audio API 合成双音符上扬旋律
  - 愉悦的开始感，增强用户体验
- 🔊 新增「断线重连成功音效」反馈 - 网络恢复时的即时反馈
  - 三音符舒缓旋律，连接恢复感
  - 网络恢复时自动播放
- 🎭 新增语音命令「你是谁」- 让AI自我介绍当前人设
  - 支持关键词：你是谁、介绍一下、自我介绍
  - AI会用符合人设的语气介绍自己
- 🔊 语音命令总数扩展至 32 种
