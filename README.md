## 📖 使用说明

1. 点击「开始对话」按钮
2. 授权摄像头和麦克风权限
3. 开始和 AI 实时对话
4. AI 会自动观察画面并主动交流

### 模式选择

- **视频双工模式** - AI 实时观察摄像头画面，适合大多数场景
- **音频双工模式** - 仅语音对话，更节省流量

### 自定义设置

点击「设置」按钮可配置：
- AI 人设提示词
- API 端点（可自建服务）
- 语音风格

## 🛠 技术架构

```
Browser (WebRTC)
    ↓ 音视频流
WebSocket Client
    ↓ JSON 消息
MiniCPM-o 4.5 API
    ↓ 实时推理
AI 响应 (语音+文字)
```

### 技术栈

- **前端**: 原生 JavaScript + WebRTC + WebSocket
- **API**: MiniCPM-o 4.5 Realtime API
- **部署**: GitHub Pages / 静态托管

### 协议说明

MiniCPM-o Realtime API 采用 WebSocket 全双工协议：

- 上行音频: 16kHz float32 PCM
- 下行音频: 24kHz float32 PCM  
- 视频帧: JPEG base64
- 消息格式: JSON

详细协议见: [API 文档](https://api.modelbest.cn/minicpmo45/docs)
